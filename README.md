# 📝 To-Do App

A simple and clean To-Do application built using **HTML, CSS, and JavaScript**.
This app helps users manage daily tasks efficiently with an easy-to-use interface.

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✔️ Mark tasks as completed
* ⌨️ Add task using Enter key
* 💾 Data saved using LocalStorage (tasks remain after refresh)
* 🎨 Clean and minimal UI

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

